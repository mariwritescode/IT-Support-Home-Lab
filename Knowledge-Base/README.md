# Knowledge Base

## Overview

This section contains Knowledge Base (KB) articles created to support common Help Desk procedures and troubleshooting tasks within the Rocky Pet Supply IT environment.

Knowledge Base articles are used to document repeatable solutions, improve consistency, reduce ticket resolution times, and provide reference material for IT support personnel.

## Available Articles

### KB-001: Restoring Access to a Shared Folder

Documents the process for verifying user group membership, reviewing NTFS permissions, and restoring access to departmental file shares.

### KB-002: Mapping a Network Drive

Provides step-by-step instructions for mapping a network drive to a shared folder and verifying successful connectivity.

### KB-003: DNS Troubleshooting for Domain Resources

Documents basic DNS troubleshooting techniques, including verifying DNS settings, flushing the DNS cache, and validating name resolution using nslookup.

## Purpose

The Knowledge Base supports the Help Desk ticket scenarios documented within this project by providing reusable troubleshooting procedures and resolution steps.
